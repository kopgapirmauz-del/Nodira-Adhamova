# Nodira Adhamova Landing

Bitta sahifali shaxsiy brend landing sahifasi. Forma orqali qoldirilgan leadlar Telegram botga yuboriladi.

## Ishga tushirish

`index.html` faylini brauzerda oching.

## Telegram sozlash

`script.js` ichida quyidagilarni almashtiring:

```js
const TELEGRAM_BOT_TOKEN = "YOUR_BOT_TOKEN";
const TELEGRAM_CHAT_ID = "YOUR_CHAT_ID";
```

Footerdagi sayt yaratgan odam linki uchun `index.html` ichidagi `YOUR_TELEGRAM_USERNAME` qiymatini o‘zingizning Telegram username bilan almashtiring.

Eslatma: token frontend kodida turgani uchun public saytda ko‘rinadi. Ishlab chiqarish uchun lead yuborishni backend yoki serverless function orqali qilish xavfsizroq.
